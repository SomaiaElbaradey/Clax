## Prerequisites
1. MongoDB
2. Redis
3. NodeJs
4. Your preferred editior. (VsCode / Android Studio)

## Installation
1. install packages  
```bash
npm install
```
2. Run the project  
```bash
npm start
```
- Note: the project can't run withot having the environment variables via author
